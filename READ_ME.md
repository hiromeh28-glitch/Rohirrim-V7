# ⚒️ ROHIRRIM | FORGE-MASTER (V11.0)
**Sacred Profit Inscription & Alchemical Logistics Engine**

The **Rohirrim Forge-Master** is a specialized logistics suite designed for high-tier crafters within the Albion Online ecosystem. It provides real-time profit analytics, resource return calculations (RRR), and secure cloud-based pattern storage.

---

## 🛡️ CORE CAPABILITIES
* **Rite of Calculation:** Instant Net Profit/Loss analysis for Gear, Alchemicals, and Provisions.
* **RRR Modulation:** Dynamic toggles for Royal, Focus, and Outlands Resource Return Rates.
* **The Grand Archive:** A secure Google Sheets backend for user authentication and pattern persistence.
* **Temporal Seal:** Integrated access expiration logic (Subscription-based access control).

---

## ⚙️ ARCHITECTURE
This suite operates on a **Zero-Footprint, Utility-First** architecture:
1.  **Frontend:** HTML5 / Tailwind CSS / JavaScript (Hosted via GitHub Pages).
2.  **Backend:** Google Apps Script (GAS) acting as a secure API gateway.
3.  **Database:** Google Sheets (The "Sacred Ledger") for user and pattern data.

---

## 🚀 DEPLOYMENT GUIDE
To clone this Forge for your own guild:
1.  **Archive Setup:** Create a Google Sheet with a tab named `Users`.
2.  **Logic Inscription:** Deploy the provided `Code.gs` via Google Apps Script as a Web App (Access: Anyone).
3.  **Link the Vault:** Replace the `FORGE_VAULT_URL` in `index.html` with your Deployment URL.
4.  **Ignite the Forge:** Upload the files to GitHub and enable **GitHub Pages** in the settings.
